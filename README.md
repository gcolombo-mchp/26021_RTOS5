<h1>Microchip MASTERs Class 26021 - RTOS5</h1><br><h2>Title:</h2><br>Mastering USB communication: Develop USB Device applications with Zephyr's USB Device Stack on your custom board<br><h2>Abstract:</h2><br>USB is now a standard serial communication channel to connect embedded systems to PCs. Upon completion of the course, you will be able to create an enumerable USB device and transfer data between a Microchip MCU and a PC using Zephyr's USB Device Stack. The hands-on part of this class is based on a development board that is not yet supported by Zephyr tree, and you will also learn how to add your own board to a Zephyr based application.
